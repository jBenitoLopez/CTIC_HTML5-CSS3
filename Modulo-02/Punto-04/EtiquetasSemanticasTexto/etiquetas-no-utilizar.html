Formato de texto con etiquetas: NO utilizar

<font>, <marquee>, <blink>,<b>,<i>, <big>, <small> o <strike>

El motivo es el mismo que ya hemos visto en el punto anterior: el HTML de una página debe indicar la estructura del contenido, no el comportamiento o aspecto del mismo. Configurar su aspecto es una tarea de CSS, por lo que debemos utilizar este lenguaje para ello, y obviar el uso de etiquetas que no sean semánticas, es decir, que no aporten conocimiento de la estructura del documento.
