Etiquetas semánticas

Énfasis y texto destacado

• <i></i>: texto en cursiva o "itálicas". El texto incluido entre esta pareja de etiquetas se visualiza inclinado y ayuda a distinguir ciertas frases dentro de un párrafo.

• <b></b>: texto en negrita (o "bold" en inglés de ahí la letra utilizada). Se emplea para mostrar en negrita el texto contenido dentro de las etiquetas.

<em></em>: se usa para enfatizar una determinada palabra o frase en el texto. Su aspecto por defecto es el de letra cursiva.

<strong></strong>: se emplea para destacar de manera prominente (fuerte) una palabra o frase en el texto. Su aspecto por defecto es el de letra en negrita.

Correcciones e inserciones

• <del></del>: se utiliza para indicar que el texto que rodea se ha eliminado del texto original. Por defecto los navegadores lo muestran como un texto tachado, pero se puede ocultar con CSS para que lo procesen las máquinas pero no lo vean los usuarios.

• <ins></ins>: su función es indicar que el texto que rodea es una nueva inserción que no estaba en el texto original de la página. Los navegadores lo suelen mostrar con una línea por debajo, como un subrayado. Se puede retirar esa línea con CSS para evitar que lo vean las personas, pero que siga indicando que es una inserción nueva.

Cite y datetime en las etiquets del y ins

cite: una dirección URL en la que se explica con más detalle los motivos del cambio.

datetime: una fecha con el formato "YYYY-MM-DDThh:mm:ssTZD" en la que se indica el día y la hora (opcional) en la que se realizó el cambio en cuestión. En el formato se incluye el año, el mes, el día, la hora, los minutos, los segundos y finalmente la zona horaria indicada por el desfase horario respecto a GMT. Por ejemplo: "2013-07-04T17:52:00+02:00" sería el 4 de Julio de 2013 a las 15:52:00 en GMT (pero el autor estaba en la zona horaria GMT+2, o sea que en esa zona horaria serían las 17:52:00).

Citas

<blockquote></blockquote>: se utiliza para indicar que el texto contenido entre las dos etiquetas es una cita de otro texto externo. Se visualiza por defecto con el párrafo sangrado a la izquierda, es decir, más separado del borde. Se puede indicar una URL en su atributo cite, como en el caso anterior, para indicar el origen de la cita. Estas etiquetas provocan un cambio de línea (son un bloque).
<q></q>: indican que el texto encerrado es una cita. La diferencia con el anterior es que se suele utilizar para citas cortas, dentro de una frase más larga (en la misma línea) y que además, por defecto, los navegadores las visualizan con unas comillas alrededor. Se pueden incluir los atributos cite y datetime.

Abreviaturas, acrónimos y definiciones.

Cuando utilizamos una abreviatura o un acrónimo en el texto podemos indicarlo mediante las etiquetas . Su atributo title sirve para manifestar su significado y se utilizará para mostrarlo en una etiqueta flotante (también conocida como tooltip) cuando el ratón pase por encima:

La palabra HTTP es el protocolo utilizado en las páginas web.

De manera idéntica se comporta la etiqueta <dfn></dfn> que sirve para marcar la posición de una palabra que vamos a definir, poniendo su definición en el atributo title.

Código

<button type="button" onclick="alert('Esto es un cuadro de alerta')">Probar alerta</button>

Nota: Ten en cuenta que si pones dentro de esta etiqueta <code> otras etiquetas HTML, éstas se interpretarán y se visualizarán, es decir, no se verá el código HTML sino su resultado. Esta etiqueta no impide que se interprete el código HTML sino que se limitan a mostrar lo que tengan dentro respetando el formato. Si quieres visualizar código HTML tienes que sustituir el "<" de las etiquetas por un "&lt;" y el ">" por "&gt;" para conseguirlo.
