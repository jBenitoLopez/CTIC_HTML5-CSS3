Prácticas propuestas para este módulo

Ejercicio 1

Crea tu primera página web partiendo absolutamente de cero. Para ello crea un archivo de texto vacío, cámbiale la extensión a .htm o .html y edítalo con el bloc de notas o un editor de texto plano similar. No hagas nada complicado: simplemente incluye los elementos estructurales básicos y un par de encabezamientos y párrafos, casi un "Hola Mundo" y poco más.
