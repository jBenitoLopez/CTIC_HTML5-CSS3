Otros atributos de enlaces

Atributo target

Si usamos un nombre cualquiera, se abrirá una nueva ventana a la cual se le dará ese nombre. Si posteriormente abrimos otro enlace que tenga el mismo nombre en el atributo target entonces se abrirá en la misma ventana:

Formación on-line - target="campusmvp"

Para lograr que un enlace se abra en una ventana aparte podemos escribir simplemente esto:

Formación on-line - target="_blank"
